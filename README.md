# Panama Canal Procurement Scraper

## What it does
Automated intelligence system that monitors and extracts 
procurement opportunities from the Panama Canal Authority, 
delivering structured, actionable data for business 
development teams.

## Business Impact
- Reduced manual research time from hours to minutes
- Delivered structured JSON output integrated into n8n 
  automation workflows
- Iterated through 153 versions to achieve production stability

## Technical Stack
- Python 3.13
- BeautifulSoup4 for HTML parsing
- Requests for HTTP/API communication
- Pandas for data processing and Excel output
- ZenRows API for anti-bot bypass
- openpyxl for Excel report generation

## How to use
1. Add your ZenRows API key to the configuration section
2. Place your input Excel file in the `entrada` folder
3. Run the script
4. Find results in the `salida` folder
